Syncing with Syncthing

Syncthing is definitely one of the things I set up as quickly as I can on a new device. I use it for:

  • syncing wallpapers between desktop devices,
  • making automatic backups of pictures on my phone,
  • keeping music files in sync between all devices (phone and PCs),
  • maintaining one todo.txt file that I can access everywhere,
  • having my savegames synced across devices, so that I can play on any one of them and have the progress propagated to other machines.

I have set up different refresh intervals for these files, depending on their importance to me. So files like wallpapers or camera backups are checked once an hour and if devices that should share them are online, they get synced. This is done opportunistically and sometimes takes a lot of time, which is absolutely fine.

One drawback of P2P, local-only system like this is the possibility of conflicts when the state of the same file changes independently on multiple devices. To mitigate this issue I set up Syncthing on an always-on Raspberry Pi Zero W, which makes the crucial files (like the TODO list) always available to other devices.
